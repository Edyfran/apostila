2. Variáveis e Tipos de Dados

Criando Variáveis

Variáveis são espaços na memória para armazenar dados:

Exemplo 4: Variáveis

Explicação:

  • Em Python, você não precisa declarar o tipo da variável. O tipo é determinado pelo valor atribuído.
  • A atribuição é feita com o operador =
  • Os nomes de variáveis devem começar com uma letra ou sublinhado e podem conter letras, números e sublinhados.
  • Python diferencia maiúsculas de minúsculas em nomes de variáveis (nome e Nome são diferentes).

Tipos de Dados Básicos

Python possui vários tipos de dados embutidos:

Exemplo 5: Tipos de Dados

Explicação:

  • int: Números inteiros, sem parte decimal.
  • float: Números com parte decimal.
  • str: Sequências de caracteres (texto).
  • bool: Valores lógicos (True ou False).
  • list: Coleções ordenadas e mutáveis de itens.
  • tuple: Coleções ordenadas e imutáveis de itens.
  • dict: Coleções de pares chave-valor.
  • A função type() retorna o tipo de uma variável.

É possível converter valores entre diferentes tipos:

Exemplo 6: Conversão de Tipos

Explicação:

  • int(): Converte para inteiro (quando converte de float, trunca a parte decimal).
  • float(): Converte para número de ponto flutuante.

3. Marcadores para Composição de Strings

Concatenação de Strings

Uma forma básica de combinar strings:

Exemplo 7: Concatenação

Explicação:

  • O operador + concatena (junta) strings.
  • Para incluir espaços entre as palavras, precisamos adicioná-los explicitamente.
  • Não podemos concatenar strings com números diretamente. Precisamos converter o número para string primeiro usando str().

Formatação de Strings com %

Um método mais antigo, mas ainda útil:

Exemplo 8: Formatação com %

Explicação:

  • %s: Placeholder para strings
  • %d ou %i: Placeholder para inteiros
  • %f: Placeholder para números de ponto flutuante
  • %.2f: Controla o número de casas decimais (2 no exemplo)
  • %05d: Preenche com zeros à esquerda até atingir 5 dígitos
  • Após a string de formatação, usamos % seguido de uma tupla com os valores na ordem correta.

Método format()

Uma maneira mais moderna de formatar strings:

Exemplo 9: Método format()

Explicação:

  • O método format() substitui cada par de chaves {} pelo argumento correspondente.
  • Podemos especificar a ordem dos argumentos usando índices dentro das chaves.
  • Podemos usar nomes para os argumentos, o que torna o código mais legível.
  • É possível controlar a formatação, como número de casas decimais {:.2f}.
  • Podemos controlar o alinhamento e preenchimento do texto com {:>10} (direita), {:<10} (esquerda) ou {:^10} (centro).

f-strings (Python 3.6+)

A forma mais moderna e recomendada de formatar strings:

Exemplo 10: f-strings

Explicação:

  • f-strings são declaradas colocando f antes das aspas.
  • Entre chaves {}, você pode colocar variáveis ou expressões que serão avaliadas.
  • É possível formatar os valores usando o mesmo formato do método format().
  • Você pode executar funções e operações diretamente dentro das chaves.
  • f-strings são mais eficientes e legíveis que os métodos anteriores.
  • Recomendado para Python 3.6 e versões mais recentes.

4. Entrada e Saída de Dados

Saída com print()

Vamos explorar mais a fundo a função print():

Exemplo 11: Função print()

Explicação:

  • A função print() é usada para exibir informações no console.
  • Pode exibir múltiplos valores separados por vírgula.

  • Strings de múltiplas linhas podem ser definidas com três aspas.
